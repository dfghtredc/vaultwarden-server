# Vaultwarden Server

Self-hosted password manager project using Vaultwarden.

## Purpose

This project is part of my self-learning portfolio in:

- Linux
- homelab infrastructure
- Docker
- web services
- reverse proxies
- cloud deployment

## Stack

- Vaultwarden
- Docker Compose
- Nginx
- Azure VM
- Cloudflare DNS
- Let's Encrypt

## Notes

- Vaultwarden runs locally on the VM
- Nginx will handle public traffic later
- secrets are stored outside GitHub
